# literature-deep-reading
功能介绍 / Function Introduction
🇨🇳 中文介绍
Literature Deep Reading & Research Mentor 是一个通用、高质量的 AI 文献精读与科研导师技能（Skill）。它从根本上解决了通用大模型在面对学术论文时“机械式摘要复述”的痛点。该技能将 AI 转化为一位严厉而专业的资深学术导师兼同行评审，通过严格的三阶段状态机（Stage Control State Machine）控制流，引导研究者深入吃透论文的本质结构与核心机制。

🛠️ 核心功能特性：
全景宏观骨架搭建（阶段一）：深度剖析论文的核心学术与产业价值；精准预警读懂该论文不可或缺的前置理论和数学工具；通过对比矩阵清晰展现本文方案与先前基线的本质差异与潜在代价。

机制深挖与证据穿透（阶段二）：严审实验设计、数据集切分及基线选择的合理性，拒绝虚高指标；逐一穿透关键图表背后的定量实际意义；解构消融实验（Ablation Study）以理解设计动机；并以“直觉-公式-压力测试”三步法拆解复杂数学与算法，探明其失效边界。

知识内化与批判复现（阶段三）：通过定制化“费曼自测”检验理解盲区；切换至严苛审稿人视角挑出论文未明说的局限性、脆弱点与潜在漏洞；提炼可直接迁移至自身课题的建模技巧；并输出定量的、模块化的代码复现路线图。

零幻觉与严谨学术规范：强制采用中英文核心术语对照，确保概念精准；无数据铁证绝不凭空捏造任何细节；在输出中严格区分作者的“明确宣称”、“合理推论”与“投机猜想”。

🇺🇸 English Introduction
Literature Deep Reading & Research Mentor is a universal, high-quality AI skill engineered for intensive academic literature reading and research mentoring. It directly addresses the common pain point where generic LLMs mechanically paraphrase abstracts without true comprehension. By transforming the AI into a rigorous senior academic mentor and peer reviewer, this skill utilizes a strict 3-Stage State Machine to guide researchers through the true intellectual, structural, and empirical depth of any paper.

🛠️ Core Functional Features:
Macro Skeleton Mapping (Stage 1): Critically analyzes the paper's genuine academic bottlenecks and industrial values; flags critical prerequisite knowledge gates to prevent the reader from getting stuck; and highlights the core differences and underlying tradeoffs against prior baselines using a structured comparative matrix.

Mechanism Deep Dive & Evidence Auditing (Stage 2): Scrutinizes experimental design, dataset splits, and baseline validity to flag weak benchmarks; decrypts the quantitative practical meaning behind primary figures; analyzes Ablation Studies to decode architectural motivations; and builds mathematical/algorithmic intuition via an "intuition-formulation-stress test" framework to discover failure boundaries.

Knowledge Internalization & Critical Review (Stage 3): Generates tailored Feynman self-test questions to expose remaining cognitive blind spots; adopts a hostile reviewer perspective to uncover unstated limitations, hidden vulnerabilities, and biases; extracts highly transferable modeling tricks for custom research workflows; and provides a concrete, step-by-step code reproduction roadmap.

Zero-Hallucination & Epistemic Rigor: Enforces mandatory bilingual terminology anchoring for conceptual precision; strictly prohibits the fabrication of absent details; and explicitly segregates explicit authors' claims from empirical inferences and speculative assertions.
